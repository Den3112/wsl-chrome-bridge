# Contributing to WSL-Chrome-Bridge

First off, thank you for considering contributing to this bridge! It's people like you that make the WSL community great.

## How to Contribute
1. **Report Bugs**: Open an Issue if you find a connection error or compatibility issue.
2. **Suggest Enhancements**: We welcome ideas on how to make the bridge faster or more automated.
3. **Submit PRs**: If you have a fix, please open a Pull Request!

## Development
- The Python proxy is in `windows/`.
- The WSL scripts are in `wsl/`.
- Use `setup.sh` to test your changes locally.

## Community & Comments
Feel free to use the **Discussions** tab for general questions or to share how you are using the bridge!
